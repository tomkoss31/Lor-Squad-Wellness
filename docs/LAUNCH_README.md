# 🚀 Lor'Squad Wellness — Premier pas

**Pour Mélanie + équipe distri · Launch local 30 avril 2026**

---

## 1. Connexion

1. Ouvre **lor-squad-wellness.vercel.app** sur ton navigateur (PC ou téléphone)
2. Connecte-toi avec ton email + mot de passe Lor'Squad
3. **Installe l'app PWA** : sur iPhone → bouton "Partager" → "Sur l'écran d'accueil". Sur Android → menu Chrome → "Installer l'app". Sur PC → icône ⊕ dans la barre d'adresse Chrome.

> 💡 **Pourquoi installer la PWA ?** Tu reçois les notifications push (RDV imminent, message client, digest matin 7h), tu lances en 1 clic depuis l'écran d'accueil, et l'app fonctionne même en offline pour consulter tes fiches.

## 2. Les 5 onglets principaux

| Onglet | Quand y aller |
|---|---|
| 🏠 **Co-pilote** | Le matin avec ton café — voir RDV du jour, suivis à faire, jauge PV, alertes |
| 📅 **Agenda** | Pour planifier un RDV, voir les prospects à recontacter |
| 💬 **Messagerie** | Lire les messages clients, leurs questions, demandes de produits |
| 👥 **Dossiers clients** | Voir tous tes clients, filtrer par statut, ouvrir une fiche |
| 💰 **Suivi PV** | Vue détaillée mensuelle, kanban des cures, plan d'action |

📚 **Centre de formation** + 🎓 **Academy** : tutos interactifs (fais-les en premier !)

## 3. Workflow type d'un nouveau client

```
1. Click "+ Nouveau bilan" (CTA gold en bas mobile / sidebar PC)
   ↓
2. Remplir 14 étapes (info client → habitudes → bodyscan → programme → félicitations)
   ↓
3. "Enregistrer et terminer" → Page remerciement plein écran
   ↓
4. Le client scanne le QR code → son app personnelle s'installe
   ↓
5. Toi tu vois le client dans /clients avec son programme + ses RDV
```

**Astuce mode démo** : bascule en mode sombre juste avant la page remerciement = effet wahou pour le client.

## 4. Mes rituels quotidiens

### Matin (5 min)
1. Ouvrir Co-pilote
2. Lire la jauge PV (rouge / ambre / vert)
3. Regarder les 3 mini-stats (clients actifs / RDV semaine / conversion)
4. Cliquer sur la carte "Suivi à faire" pour voir mes 3 priorités du jour

### Après chaque RDV (30 sec)
1. Sur la fiche client, clic onglet "Actions"
2. Choisir le statut lifecycle (actif / en pause / arrêté / etc)
3. Planifier le prochain follow-up

### Hebdo (5 min, dimanche soir)
1. Aller sur /pv
2. Regarder ma jauge mensuelle
3. Ouvrir le **Plan PV** (section card) — voir les 3 clients prioritaires à relancer
4. Planifier les relances de la semaine
